## A. 전체 구조 비교

| 구분 | 현재 프로그램 (myjae) | 시중 일반 (천기대요式) | 노트북LM 자료 (심산강의·통변요론) |
|---|---|---|---|
| 결혼택일 기능 | **사실상 없음** — 제목만 "결혼 택일 결과", 실제론 궁합 분석(SajuSummary·AreaAnalysis)을 보여줌 | 날짜 스캔 → 흉일 거르고 길신일 추천 | 0단계 필터(불가일 제외) + 1단계 100점 채점 |
| 점수 체계 | 없음 (prewedding은 점수판도 숨김) | 길신 개수 많을수록 좋음 (정성적) | **100점 만점 정량 배점** |

## B. "피하는 날"(거르기) 비교

| 흉일 항목 | 현재 프로그램 | 시중 일반 | 노트북LM 자료 |
|---|---|---|---|
| 남녀 공망일 | ✖ (택일 안 함) | △ 일부 | ✔ 0단계 제외 (남녀 일주 기준) |
| 일지 원진·충 | ✖ | ✔ 상충·형살 회피 | ✔ 0단계 제외 (子未·丑午… 원진 / 子午·丑未… 충) |
| 사폐일 | ✖ | △ 천기대요에 포함 | ✔ 하드코딩 (봄庚申·여름壬子·가을甲寅·겨울丙午) |
| 천지전일 | ✖ | △ | ✔ 하드코딩 (계절별 간지) |
| 혼인총기일(화해·절명·홍사·수사·천적 등) | ✖ | ✔ 핵심 (천기대요 혼인총기일) | (자료엔 일부만, 시중이 더 광범위) |
| 본명일(남녀 일진과 같은 날) | ✖ | ✔ | ✔ (원진·충에 포함 개념) |
| 복단일·월파·고신·과숙 | ✖ | ✔ | △ |
| 절기 전환일(입춘·하지 등) | ✖ | ✔ | △ |

> 정리: 시중은 흉일 종류가 매우 많음(천기대요 전체). 노트북LM은 그 중 **핵심 5종을 정량 0점 처리**로 단순·명확하게 정리. 현재 프로그램은 전무.

## C. "좋은 날"(가점) 비교

| 길신 항목 | 현재 프로그램 | 시중 일반 | 노트북LM 자료 (배점) |
|---|---|---|---|
| 천을귀인일 | ✖ | △ 길신 중 하나 | ✔ **30점** (남녀 일간 기준, 둘 다면 최고) |
| 부부 용신일 | ✖ | △ 오행 보완 개념 | ✔ **30점** (남녀 용신 오행이 그날 간지에) |
| 손 없는 날 | ✖ | ✔ (음력 9·10·19·20·29·30) | ✔ **20점** (음력 끝자리 9·0) |
| 천희성·홍란성 | ✖ | △ | ✔ **20점** (년지 기준 도출 + 일지와 합) |
| 천덕일·월덕일 | ✖ | ✔ **핵심 길신** | (자료엔 명시 적음) |
| 황도일·음양부장길일·십전대길일 | ✖ | ✔ | △ |
| 오행 균형 교집합 | ✖ | ✔ (두 사람 부족 오행 보완) | ✔ (용신일에 포함 개념) |

> 정리: 시중은 천덕·월덕·황도 등 전통 길신 중심. 노트북LM은 **천을귀인·용신·손없는날·천희홍란 4종에 명확한 배점**(30/30/20/20). 우리 구현엔 노트북LM 배점을 기본 골격으로 쓰되, 천덕·월덕(시중 핵심)을 보강하면 가장 탄탄함.

## E. 출산택일 vs 결혼택일 (구조 대응)

| | 출산택일 (완성) | 결혼택일 (만들 것) |
|---|---|---|
| 기간 | 예정일 ±1주 (좁음) | 사용자 지정 기간 (예: 가을~내년 봄) + 주말 위주 |
| 거르기 | 산액 회피(子卯酉·충·쏠림) | 0단계: 공망·원진·충·사폐·천지전 |
| 채점 | 조후30·오행30·지지40 | 천을귀인30·용신30·손없는날20·천희홍란20 |
| 중심 | 아기 사주 | 두 사람(남녀) 사주 + 그날 |
| 결과 | 추천5 + 피할날 | 추천 길일 + 피할날 |

> 핵심 원칙(시중 공통): **한쪽에만 좋은 날보다 두 사람 모두에게 부담 적은 날 우선.**
> 택일 ≠ 궁합. 궁합이 먼저고, 택일은 "언제 식을 올릴지" 돕는 도구.

## F. 권장 구현 방향 (결론)

1. **골격은 노트북LM 자료**(0단계 필터 + 100점 배점)를 그대로 사용 — 정량적이라 코딩에 바로 대입 가능.
2. **시중 핵심 길신(천덕·월덕)**을 가점에 보강 — 전통 정통성 강화.
3. **두 사람 모두 기준**으로 — 한쪽만 좋은 날은 감점(시중 공통 원칙).
4. 기준일은 **예식일**. 후보는 **주말 위주**로 좁힘.
5. 점수 기준은 **잠정** — 연재 선생님 검수 후 확정.
