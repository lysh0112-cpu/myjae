# 이달의 운세 100점 만점 산식

> 코드: `lib/saju/monthlyFortune.ts` · 등급표: `lib/saju/jijiGrade.ts`
> 화면: `app/manseryeok/components/TodayFortuneCard.tsx` (탭 방식)
> ※ 심산 명리학에서 **월운을 점수화한 전례가 없습니다.** 소스에는 지지 관계 등급(A~D)과
>    영향력 순위만 있고, 100점으로 옮기는 기준은 없습니다. → 명카페가 정한 표시 기준.
>    명리 판단이 필요한 부분(등급표, 월지·일지 비중)은 연재쌤 확인을 거쳤습니다.
> 작성일: 2026-07-20

---

## 0. 한눈에 보기

내 사주(일간·월지·일지)와 이달 월건(천간·지지)의 관계를 네 항목으로 채점합니다.

```
총점 = ① 오행·용신 (30점)
     + ② 천간 관계 (20점)
     + ③ 지지 관계 (30점) ← 월지 20 + 일지 10
     + ④ 십성       (20점)
     ─────────────────
                  100점
```

### 오늘의 운세와 다른 점

- **월지를 함께 봅니다.** 오늘의 운세는 일지만 보지만, 월운은 월지(20점) + 일지(10점)
- **144칸 등급표를 씁니다.** 합·충만 보는 게 아니라 연재쌤 책의 A~D 등급표를 그대로 적용
- **신살 대신 십성**을 봅니다. 한 달 단위라 공망은 적용하지 않습니다

## 1. ① 오행·용신 (30점)

이달 천간의 오행과 이달 지지의 오행을 각각 판정해 더합니다. 각 **최대 16점**.

| 이달 오행이 | 점수 | 뜻 |
|---|---|---|
| 내 **용신**과 같음 | **16** | 꼭 필요한 기운 |
| 내 **희신**과 같음 | 14 | 도움이 되는 기운 |
| 그 밖 (한신·구신) | 12 | 보통 |
| 내 **기신**과 같음 | 10 | 부담되지만 바닥은 아님 |

천간 + 지지 = 최소 20점, 최대 32점. 30점 만점 항목이라 실제로는 20~30점 범위입니다.

|---|---|---|

## 3. ③ 지지 관계 (30점) — 핵심

**이달 지지**를 내 **월지**와 **일지**에 각각 대입해 등급을 뽑고, 배점을 나눠 더합니다.

```
지지 점수 = 월지 반응 (20점 만점) + 일지 반응 (10점 만점)
```

### 왜 월지가 20점인가

『심산 명리비법 적성노트』 84쪽

> "대운이나 세운을 일단 月支에 대입해라. **月支가 총사령관이다**"

85쪽 영향력 순위

> **月支 > 日支 > 時支 > 年支**

### 등급 → 점수 환산

| 등급 | 비율 | 월지 (20점) | 일지 (10점) | 뜻 |
|---|---|---|---|---|
| **A** | 100% | **20.0** | **10.0** | 매우 좋음 |
| **B** | 88% | 17.6 | 8.8 | 좋음 |
| **C** | 76% | 15.2 | 7.6 | 보통 |
| **D** | 62% | 12.4 | 6.2 | 주의 |

> 소스(강의노트 201쪽)의 `A 3점 / B 1.5점 / C 0점 / D −1.5점` 은 **가점·감점 방식**이라
> 100점 만점에 그대로 넣으면 하한이 너무 낮아집니다. 비율만 참고해 위 값으로 잡았습니다.

### 144칸 등급표

등급은 `lib/saju/jijiGrade.ts` 의 **12지지 × 12지지 = 144칸** 표에서 가져옵니다.
출처는 『심산 명리비법 적성노트』 49쪽 요약표 + 51~73쪽 상세 해설.

#### 이 표의 특징 — 대칭이 아니다

같은 관계라도 **내 지지가 무엇이냐에 따라 등급이 다릅니다.**

| 내 지지 + 운 | 등급 | 소스 설명 |
|---|---|---|
| 子 + 丑 | A | 子丑合 — 코드가 맞는 사적인 결합, 대인 관계가 무난 |
| 丑 + 子 | B | 子丑合이지만 **한랭** — 원국에 木火가 있을 때만 좋음 |

표에는 합·충뿐 아니라 **격각·암합·입묘·귀문·가합·방국** 같은 관계가 모두 들어 있습니다.
각 칸마다 연재쌤 책의 해설 문장이 함께 담겨 있어 화면과 AI 프롬프트에 그대로 쓸 수 있습니다.

#### 확정된 3칸 (2026-07-19)

49쪽 요약표와 상세 해설의 등급이 달랐던 세 칸은 연재쌤이 아래로 확정하셨습니다.

- **未寅 = D** (寅未귀문)
- **戌辰 = B** (辰戌沖)
- **戌戌 = D** (해외 역마)

셋 다 상세 해설 쪽 값입니다.

## 5. 등급 구분

| 점수 | 등급 이름 | 비율 | 뜻 |
|---|---|---|---|
| 93 ~ 100 | **활짝 열리는 달** | 2.5% | 크게 나아갈 수 있는 달 |
| 87 ~ 92 | **순한 바람이 부는 달** | 23.0% | 일이 순조롭게 풀리는 달 |
| 75 ~ 86 | **차곡차곡 쌓는 달** | 70.0% | 꾸준히 준비하는 달 |
| 67 ~ 74 | **숨 고르는 달** | 4.5% | 서두르지 않고 정리하는 달 |

### 월운 등급을 후하게 잡은 이유

월운은 **한 달 내내 같은 점수가 화면에 떠 있습니다.** 하루짜리 운세와 달리
낮은 등급이 계속 보이면 사용자가 앱을 떠나게 됩니다.
그래서 **"숨 고르는 달"은 4.5%** 로 아주 적게 나오도록 경계를 잡았습니다.

## 7. 계산 예시

**류도이님** (일간 壬 · 월지 子 · 일지 子 · 용신 木) — 2026년 7월 乙未월

| 항목 | 판정 | 점수 | 이유 |
|---|---|---|---|
| ① 오행·용신 | 乙=木(용신) | 28 / 30 | 천간 16 + 지지 12 |
| ② 천간 | 내가 생함 | 16 / 20 | 壬(水)이 乙(木)을 생함 |
| ③ 지지 | 둘 다 D | 18.6 / 30 | 월지 子未원진 12.4 + 일지 子未원진 6.2 |
| ④ 십성 | 식상 | 17 / 20 | 水가 木을 생하므로 식상 |
| **총점** | | **80 / 100** | **차곡차곡 쌓는 달** |

이 사주는 월지·일지가 모두 子이므로 두 영역 결과가 같습니다.
화면에서는 **한 줄로 합쳐** 표시됩니다.

## 8. 화면에 함께 나오는 것

점수 외에 **AI 호출 없이 계산만으로** 만들어지는 정보들.

| 항목 | 내용 |
|---|---|
| **일·환경 / 나·건강** | 월지 반응과 일지 반응을 나눠 막대로 표시. "일은 잘 풀리는데 몸이 따라주지 않는 달" 같은 해석 가능 |
| **지난달 대비** | 이번 달 점수와 지난달 점수의 차이 |
| **올해 흐름** | 12개월 점수를 막대그래프로. "10월에 큰 흐름이 온다"를 미리 볼 수 있음 |
| **이번 달 눈여겨볼 날** | 날짜별 일진을 내 일지와 대조해 좋은 날 3개, 조심할 날 1개. **오늘 이후만** |
| **이달의 명리 한 조각** | 144칸 표의 연재쌤 책 해설을 그대로 표시. 하단에 "명리 근거: 子未원진" 출처 표기 |

## 9. 유지보수

### 조정 가능한 값

`lib/saju/monthlyFortune.ts` 상단에 모여 있습니다.

```ts
GRADE_RATIO = { A: 1.0, B: 0.88, C: 0.76, D: 0.62 }   // 등급→점수 비율
BRANCH_MAX  = { month: 20, day: 10 }                   // 월지·일지 배분
CUTS        = { S: 93, A: 87, B: 75 }                  // 등급 경계
```

### 등급 이름
같은 파일의 `MONTH_GRADE_LABEL`.

### 등급표 자체를 고치려면
`lib/saju/jijiGrade.ts` — 144칸 각각에 `grade` / `tag` / `desc` 가 들어 있습니다.
