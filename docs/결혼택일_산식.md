# 결혼택일 100점 만점 산식 (정밀 구체화)

> 노트북LM 자료(심산강의·통변요론) 기준. 코딩에 바로 대입할 수 있게 구체화.
> ※ 전통 길흉표(천을귀인·천희홍란·사폐·천지전)는 주신 자료의 공식을 그대로 따름.
>    학파별 차이 가능 → "잠정 기준", 연재 선생님 검수 후 확정.
> 작성일: 2026-06-30

---

## 0. 인덱스 정의 (코딩 기준)

천간(STEM)  : 甲0 乙1 丙2 丁3 戊4 己5 庚6 辛7 壬8 癸9
지지(BRANCH): 子0 丑1 寅2 卯3 辰4 巳5 午6 未7 申8 酉9 戌10 亥11
오행: 甲乙寅卯=목, 丙丁巳午=화, 戊己辰戌丑未=토, 庚辛申酉=금, 壬癸子亥=수

## 입력값

신랑: gStem(일간), gBranch(일지), gYearBranch(년지), gYongsin(용신 오행), 일주60갑자
신부: bStem, bBranch, bYearBranch, bYongsin, 일주60갑자
후보날: dStem(일간), dBranch(일지), 간지문자열, lunarDay(음력일), season(절기상 계절)

계절(season)은 절기 기준 월지로 판정:
- 봄=寅卯辰월, 여름=巳午未월, 가을=申酉戌월, 겨울=亥子丑월

## 0단계: 강력 필터 (하나라도 걸리면 후보 제외 → "피할 날"로 분리)

### F1. 남녀 공망일
- 신랑/신부 각 일주(60갑자) 기준 공망 지지 2개 산출(순공).
- 후보날 일지가 신랑 공망 OR 신부 공망 → 제외.
- 기존 lib/saju/gongmang.ts 재사용.

### F2. 일지 충(沖)  — 쌍: 子午 丑未 寅申 卯酉 辰戌 巳亥
- 후보날 일지가 신랑 일지 OR 신부 일지와 충 → 제외.

### F3. 일지 원진(怨嗔) — 쌍: 子未 丑午 寅酉 卯申 辰亥 巳戌
- 후보날 일지가 신랑/신부 일지와 원진 → 제외.

### F4. 본명일
- 후보날 일주(간지)가 신랑 OR 신부 일주(간지)와 완전히 동일 → 제외.

### F5. 사폐일(四廢日) — 작사불성·유시무종 대흉
| 계절 | 사폐 간지 |
|---|---|
| 봄 | 庚申 |
| 여름 | 壬子 |
| 가을 | 甲寅 |
| 겨울 | 丙午 |
- 후보날 간지가 해당 계절 사폐 간지 → 제외.

### F6. 천지전일(天地轉日) — 교통사고·요수 대흉
| 계절 | 천지전 간지 |
|---|---|
| 봄 | 乙卯, 辛卯 |
| 여름 | 丙午, 戊午 |
| 가을 | 辛酉, 癸酉 |
| 겨울 | 壬子, 丙子 |
- 후보날 간지가 해당 → 제외.

> ※ 丙午는 여름 사폐이자 여름 천지전에 모두 등장 → 어느 쪽으로든 제외되면 됨.
> ※ 壬子도 여름 사폐 / 겨울 천지전에 등장. 계절 판정 후 해당 규칙만 적용.

## 1단계: 가점 채점 (100점 만점)

0단계를 통과한 날만 채점.

### ① 천을귀인일 — 30점
| 일간 | 천을귀인 지지 |
|---|---|
| 甲·戊·庚 | 丑, 未 |
| 乙·己 | 子, 申 |
| 丙·丁 | 亥, 酉 |
| 壬·癸 | 巳, 卯 |
| 辛 | 午, 寅 |

- 후보날 일지가 신랑 일간 기준 천을귀인 → +15
- 후보날 일지가 신부 일간 기준 천을귀인 → +15
- 둘 다면 30. 한쪽만 15. 없으면 0.

### ② 부부 용신일 — 30점
- 후보날 간지 오행 집합 = { dStem오행, dBranch오행 }
- 신랑 용신 포함 → +15, 신부 용신 포함 → +15 (둘 다 30)
- 용신 모르면 0, 다른 항목으로 채점.

### ③ 손 없는 날 — 20점
- 음력일 끝자리 9 또는 0 → +20.
- /api/lunar 의 lunarDay 사용.

### ④ 천희성·홍란성 — 20점
도출 공식(자료 그대로):
- 천희지index = (9 - 년지index + 12) % 12   (酉=9에서 년지만큼 역행)
- 홍란지index = (천희지index + 6) % 12       (천희와 충)
- 신랑·신부 각각 천희·홍란 산출(년지 기준).

채점(하나라도 만족 시 +20):
- 후보날 일지 = 신랑/신부 천희성 또는 홍란성, OR
- 후보날 일지가 신랑/신부 일지와 육합 또는 삼합
  - 육합: 子丑 寅亥 卯戌 辰酉 巳申 午未
  - 삼합: 申子辰(수) 寅午戌(화) 巳酉丑(금) 亥卯未(목)

### ⑤ (보강) 천덕·월덕일 — +10 (시중 핵심 길신, 선택)
- 월지 기준 천덕/월덕 천간·지지가 후보날에 들면 +10.
- 총점 100 상한.

## 결과 출력 (출산택일과 동일 패턴)

- 0단계 통과 후보 점수순 정렬 → 상위 5개 (날짜 중복 제거, 주말 우선)
- 각 추천: 날짜(요일)·간지·점수·등급, 적용 길신 배지, AI 한 줄 해설
- 1순위 상세 해설
- "피하면 좋은 날": 0단계 탈락일 + 사유(공망/충/원진/사폐/천지전) 최대 3개
- 안내: "전통 명리 참고용 · 실제 예식일은 양가·예식장 사정과 함께 결정하세요"

## 개발 단계 (출산택일 패턴 재사용)

1. 메뉴 연결: couple-input에서 prewedding → /manseryeok/wedding-timing 분기
2. 설문 화면 (기간 시작·끝, 요일 선호)
3. 후보 날짜 생성 + 사주 뽑기 (birth-timing/candidates.ts 구조 재사용)
4. 채점: 0단계 필터 + 1단계 100점 ← 핵심
5. 정렬 5개 + 피할 날
6. 결과 화면 (birth-timing/result 구조 재사용)
7. AI 해설 (/api/analyze)
