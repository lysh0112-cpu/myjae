## 1. 사주 8글자는 어떻게 만들어지나

핵심 흐름은 `useResultSaju` 훅(`app/manseryeok/.../useResultSaju.ts`)에 있다.

1. **`/api/lunar` 를 한 번 호출** → 연·월·일 간지를 받는다.
   - 음력이면 양력 변환(solarYear/Month/Day)도 같이 돌려준다.
   - 절기·KASI 처리는 이 API가 내부에서 다 한다.
2. **시주(時)는 `calcHourPillar(일간, 시진index)` 로 계산** — API 없이 코드로.
3. 결과는 `[시주, 일주, 월주, 년주]` 순서, 각각 `{ stem(천간), branch(지지) }`.
